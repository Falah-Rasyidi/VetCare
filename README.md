## General info
VetCare allows pet-owners to easily connect with local clinics to find the best treatment for their pets. Furthermore, it serves as a hub for educational content and a one-stop shop for prescription medicine for pets.

## Technologies
* Springboot
* Thymeleaf
* Java
* HTML
* CSS
* Bootstrap
* H2

## Setup
Ensure you are in the 'codebase' directory using `cd codebase`. Then, you may run `./mvnw clean spring-boot:run` to start the application. Once the application has started running, open localhost in your browser [http://localhost:8080](http://localhost:8080) to view the website
